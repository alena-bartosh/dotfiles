# dotfiles

My set of zsh, git, and tmux configuration files.

```
$ git clone https://github.com/alena-bartosh/dotfiles.git
$ cd dotfiles
```

### Git

```
$ scripts/setup_git.sh
```

### Oh My Steps

```
$ scripts/install_zsh.sh
$ scripts/setup_ohmyzsh.sh
```
